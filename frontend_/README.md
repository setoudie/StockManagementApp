<h1 align="center">React Admin Dashboard</h1>

### This is a free admin dashboard ui kit built with:

-   React
-   Tailwind CSS
-   Recharts
-   Framer Motion

It is completely responsive and can be used for any type of web application.

### Run this app locally

```shell
npm install
```

```shell
npm run start
```
